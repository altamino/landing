We're still building out this page — for now, here are a few quick answers to common questions.

  • What is AltAmino? An open-source, from-scratch reimplementation of the original Amino server, built after the original app shut down in December 2025.
  • Is this an official Amino product? No, AltAmino is an independent, community-driven project and is not affiliated with the original Amino team.
  • I forgot my password / can't log in. See our Support page for how to get help with your account.
  • I found a bug. Please report it on our GitHub page so we can look into it.
  • The project is still in development — expect bugs, missing features, and occasional downtime while we keep improving things.

Couldn't find what you're looking for? Reach out via our Support page.
